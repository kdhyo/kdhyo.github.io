자신의 근처 이웃들에게 도움을 요청해 다른사람의 손이 필요한 일을 부탁해보세요!

기간 : 20.08월 ~ 20.10월

사용된 기능

  • GrapQL-yoga 기반 서버 셋팅
  • Prisma2 ORM을 통한 MySQL
  • nodemon을 통한 개발환경 설정
  • cors 방지
  • morgan 패키지를 활용한 로그기록
  • babel 패키지를 활용한 ES6 이상문법 및 빌드 설정
  • dotenv 패키지를 통한 .env 사용
  • jsonwebtoken을 통한 회원 토큰 생성
  • passport / passport-jwt 를 통한 회원 인증구현
  • bcryptjs / crypto 통한 패스워드 암호화기법 적용
  • nodemailer / bcryptjs 사용으로 이메일을 통한 랜덤문자 전송을 이용한 이메일인증
  • merge-graphql-schemas / path / graphql-tools 를 사용하여 디렉토리 관리
  • Graphql-yoga pubsub를 활용한 신규 서비스 및 채팅 실시간 구현
  • multer multer-s3 aws-sdk path 사용을 통한 aws s3에 서비스 이미지 업로드
  • aws RDS를 통한 MySQL 호스팅 사용
처음 사용해보는 Grapqhl을 통해 rest api와의 차이점 및 graphql의 장점들을 알 수 있는 기회가 되었습니다.
graphql과 같이 사용하기 좋은 prisma를 통해 좀 더 쉽게 mysql을 동작시킬 수 있는 걸 경험해보았으며,
처음으로 nodejs-express로 셋팅이 되어 있는게 아니라 직접 바벨, morgan 등 필요한 것들만 셋팅해보는 경험도 할 수 있었습니다.
또한, graphql-yoga에 내장되어있는 소켓.io를 경험하여 실시간으로 데이터를 받아와 실시간 서비스조회, 채팅시스템을 구축해보았으며,
aws에 있는 s3와 rds를 사용하여 기존에 경험하지 못했던 aws들도 다뤄볼 수 있었습니다.

MapStival

설명

원하는 지역의 축제를 한 눈에 볼 수 있는 사이트

기간 : 20.04월 ~ 20.06월

사용된 기능

  • nodejs-express를 활용한 백엔드 프로트타입 제작
  • axios를 활용한 API 호출
  • async/await를 활용한 축제API 비동기 처리
  • sequelize를 통한 mysql 연동
  • CRUD를 적용한 각 축제별 댓글기능 구현
  • aws를 활용한 서비스 배포
첫 프로젝트로서 Mysql, sequelize, Nodejs, expressjs, ejs 등 새로운 스킬들을 활용해볼 수 있는 기회였습니다.
axios를 활용하여 공공api들을 다양하게 이용해보면서 api에 대한 이해를 높이는 계기가 되었습니다.
비동기가 왜 필요한지, 언제 써야할지에 대해 직접 해보면서, 겪어보면서 알 수 있게 된 경험이었으며, Mysql을 연동하여 CRUD를 직접 하나하나 구현해보면서 쿼리와 서버사이드에서 전달하는 방식을 이해할 수 있게 됐습니다.
처음으로 aws를 이용하여 서비스를 배포해보는 경험을 하면서 한층 더 성장한 느낌을 받았으며, 유닉스를 배워야하는 이유와 유닉스에 대한 간단한 활용을 해볼 수 있는 좋은 기회였습니다.

ShoppingMall

설명

인프런 강좌를 보고 따라서 만들어본 쇼핑몰 사이트입니다. 인프런강좌

기간 : 20.07월

배운점

  • MONGODB 연결
  • MONGODB 함수를 활용한 데이터 CRUD
  • React Hook (State, Effect)를 활용하여 Reack 컴포넌트 효율을 높임
  • map(), forEach() 차이점
  • REST API를 활용한 Client에서 Server를 호출하여 데이터 상호전달
  • Ant Design of React를 활용한 디자인 구현
  • React-Redux에 대한 사전지식이 없는 채 시작을 하여 완벽히 이해는 하지 못했으나, 어떻게 사용하는지에 대해 간략하게 눈으로 익힘
  • axios 및 Redux-dispatch에 대한 이해 및 활용방법
NoSQL인 Mongoodb를 사용해보면서 NoSQL을 처음 접해보았습니다.
몽고db에 있는 함수들을 활용해보면서 MySQL과 다른 점들을 알 수 있게 되었으며, 몽고DB 설계 및 CRUD활용을 할 수 있게 되었습니다.
React Hook을 이론적으로만 알고 있었는데 이번 프로젝트를 통해 많은 사용을 해보면서 왜 필요한지, 왜 사용하는지를 알게 되었고, 클래스방식과 함수방식에 차이점을 알 수 있는 기회였습니다.
또한 Server와 Client를 따로 두어 RESTful을 해보면서 서버와 클라이언트의 전달방식을 접해보면서 활용방법을 알게 되었습니다.
React에 antd인 라이브러리를 활용하여 여러 디자인들을 import하여 사용해보았습니다.
리덕스에 대한 사전지식이 없이 강좌를 본 터라 리덕스를 완벽하게 이해하진 못했지만, 어떻게 사용하는지 간단하게 소스코드를 보면서 이해를 해보았습니다.
